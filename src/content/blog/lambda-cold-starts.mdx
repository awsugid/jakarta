# The Cold Start Problem

Cold starts happen when a new execution environment is spun up to handle a request.

## Valid Strategies

1. **SnapStart** for Java
2. **Provisioned Concurrency**
3. **Lighter Runtimes** (Node.js, Python, Rust)

## What Doesn't Work

Calling your function every minute is not a scalable solution.

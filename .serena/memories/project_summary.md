# Project Summary

## Purpose
Event management website for the AWS User Group Jakarta, aiming to introduce community activities, upcoming events, and facilitate collaboration with companies and communities (sponsors/partners).

## Tech Stack
- **Framework**: Astro 5.16.5 (with React integration)
- **Frontend Library**: React 19.2.3
- **Styling**: Tailwind CSS v4.1.18 (using `@tailwindcss/vite`)
- **UI Components**: shadcn/ui (New York style)
- **Package Manager**: Bun
- **Language**: TypeScript

## Structure
- `src/pages`: File-based routing (Astro)
- `src/components`: Astro and React components
- `src/components/ui`: shadcn/ui components
- `src/layouts`: Astro layouts
- `src/lib`: Utilities (`utils.ts` for `cn`)
- `src/styles`: Global styles (`global.css`)
- `public`: Static assets
